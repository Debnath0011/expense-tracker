# Expense Tracker

A full-stack Expense Tracker web application built using the MERN stack that helps users manage income and expenses, track savings, visualize spending patterns, and monitor financial activity.

## Features

- User Authentication (Login & Register)
- Add Income and Expense Transactions
- Edit and Delete Transactions
- Real-Time Balance Calculation
- Monthly Income, Expense and Savings Overview
- Recent Activity Section
- Search Transactions
- Filter by Income and Expense
- Pie Chart Visualization
- Dark Mode Support
- CSV Export Functionality
- Responsive Design

## Tech Stack

### Frontend
- React.js
- CSS3
- Chart.js
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT (JSON Web Token)

## Installation

### Clone Repository

```bash
git clone https://github.com/Debnath0011/expense-tracker.git
```

### Install Dependencies

```bash
npm install
cd client
npm install
cd ../server
npm install
```

### Run Application

Frontend:

```bash
cd client
npm run dev
```

Backend:

```bash
cd server
node server.js
```
